# Portal de Cidadania Digital e Inteligência Artificial 🌐

Projeto desenvolvido para o Componente Curricular de **Educação Digital e Computação: Programação e IA**, com foco na conscientização comunitária a respeito dos impactos de mídias manipuladas (Deepfakes) e desinformação automatizada na sociedade moderna.

## 🚀 Funcionalidades Técnicas Implementadas
- **Estruturação Semântica**: Arquitetura limpa utilizando as tags estruturais do HTML5 (`main`, `section`, `footer`, `form`).
- **Layout Moderno**: Organização visual fluida estruturada através de regras Flexbox aplicadas via CSS3.
- **Acessibilidade e Interatividade**: Script dinâmico em JavaScript Vanilla responsável pelo controle de alternância do Modo Escuro e processamento lógico com validação do Quiz.
- **Responsividade**: Adaptação completa da interface para dispositivos móveis via CSS Media Queries.

## 🤖 Prompts de Inteligência Artificial Utilizados
Para auxiliar na construção e validação ética deste portal, foram aplicados os seguintes comandos:
1. *Refinamento de Código*: "Gere uma estrutura de formulário HTML5 acessível para um quiz contendo inputs do tipo radio com labels bem associados."
2. *Estilização de Paleta*: "Sugira uma paleta de cores CSS moderna baseada em tons tecnológicos e de segurança (azul escuro e cinzas neutros) com suporte nativo a Dark Mode."
